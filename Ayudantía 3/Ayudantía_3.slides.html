cabin N√∫mero de cabina
embarked Puerto de embarque C = Cherburgo, Q = Queenstown, S = Southampton

Notas de variables


pclass: Un proxy para el estado socioecon√≥mico (SES)
1¬∫ = Upper
2do = Medio
3ro = Lower
age: Age es fraccionario si es menor que 1. Si se estima la edad, tiene la forma de xx.5
sibsp: El conjunto de datos define las relaciones familiares de esta manera...

  • Sibling = hermano, hermana, hermanastro, hermanastra
  • Spouse = marido, mujer (se ignoraron amantes y novios)

parch: El conjunto de datos define las relaciones familiares de esta manera...

  • Parent = madre, padre
  • Child = hija, hijo, hijastra, hijastro

Algunos ni√±os viajaron solo con una ni√±era, por lo tanto parch=0 para ellos.
